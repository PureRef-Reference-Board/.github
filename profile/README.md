# PureRef Reference Board

**PureRef Reference Board** is a lightweight, clutter-free application for artists, designers, and creators to collect, view, and organize visual references. As a powerful **PureRef Image Organizer**, you can drag images from any source (web, folders, or other apps) onto an infinite canvas, then freely scale, rotate, arrange, and layer them without resolution loss. This **PureRef Mood Board Tool** features always-on-top mode so references stay visible while you work in Photoshop, Blender, ZBrush, or any creative software. With **PureRef Artist Canvas**, you can save boards as standalone .pur files that embed all images, making sharing reference collections with clients or students effortless.

## Features

- **Infinite Canvas**: The **PureRef Reference Board** never runs out of space for your images.
- **Zoom Forever**: Use **PureRef Image Organizer** for vector-style zoom without pixelation.
- **Always on Top**: As a **PureRef Mood Board Tool**, it stays above other windows.
- **Embedded Images**: This **PureRef Artist Canvas** saves everything in one self-contained file.
